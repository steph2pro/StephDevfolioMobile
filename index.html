Mes Travaux

Application PerfectPay

PerfectPay est une application mobile de transactions financières, conçue pour faciliter les opérations de dépôt, de retrait et de transfert d’argent de manière rapide et sécurisée. Elle offre une interface intuitive, la gestion sécurisée des comptes, et un suivi en temps réel des transactions.

Technologies utilisées :

  • Flutter – pour une interface utilisateur fluide et performante
  • Clean Architecture – pour un code structuré et évolutif
  • BLoC Pattern – pour la gestion d’état
  • Firebase Authentication – pour une authentification sécurisée
  • Firebase Firestore – pour la gestion des données en temps réel
  • SecureStorage – pour stocker les informations sensibles (token, code PIN)
  • Code PIN – pour sécuriser l’accès à l’application
  • QR Code – génération et scan pour paiements rapides
  • CustomPainter – pour des interfaces et animations personnalisées
  • Dio – pour les requêtes HTTP
  • l10n – pour la traduction et l’internationalisation
  • AutoRoute – pour la navigation optimisée
